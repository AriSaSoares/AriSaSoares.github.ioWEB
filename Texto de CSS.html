Uma breve história sobre o CSS

O CSS (Cascading Style Sheets) foi proposto pela primeira vez por Håkon Wium Lie em 1994 e desenvolvido em conjunto com Bert Bos Antes do CSS, o estilo das páginas da web era definido diretamente no HTML, o que resultava em uma mistura confusa de estrutura e estilo. Com o crescimento da web e a necessidade de controle mais granular sobre o design das páginas, surgiu a demanda por uma linguagem dedicada para controlar a apresentação visual.

A proposta de Lie e Bos introduziu a ideia de um sistema de estilo independente, permitindo que os desenvolvedores aplicassem estilos separados para diferentes elementos em um documento HTML.

A primeira versão oficial do CSS, conhecida como CSS nível 1, foi lançada em dezembro de 1996 pelo W3C (World Wide Web Consortium). Esta versão inicial do CSS introduziu propriedades para controlar a cor do texto, fontes, margens e posicionamento.

Desde então, o CSS evoluiu através de várias versões (CSS2, CSS3) e continua a se desenvolver com novas funcionalidades, capacidades e aprimoramentos para atender às demandas crescentes do design web moderno. O CSS tornou-se uma linguagem fundamental para a web, permitindo a criação de páginas mais atraentes e funcionais, ao mesmo tempo em que separa o conteúdo da sua apresentação visual.

A função principal do CSS é separar o conteúdo estrutural (definido pelo HTML) da apresentação (definida pelo CSS). Isso permite que você aplique estilos visuais, como cores, fontes, espaçamentos, posicionamento e layouts, sem alterar a estrutura do conteúdo HTML. Dessa forma, o CSS ajuda a garantir consistência e facilita a manutenção do design de um site ou aplicativo.

PONTOS SOBRE O CSS:


  • Seletores:

O CSS usa seletores para identificar os elementos HTML aos quais se aplicarão estilos. Por exemplo, você pode selecionar elementos por tipo (tag), classe, ID ou hierarquia.

  • Propriedade e Valores

As propriedades são características que você deseja alterar, como cor de fundo, tamanho da fonte, margens, etc. Cada propriedade tem um valor associado que determina como a propriedade será aplicada.

  • Cascata e Especificidade:

O termo "cascata" no CSS se refere à maneira como diferentes estilos são aplicados e resolvidos quando há regras conflitantes. A especificidade define qual regra será aplicada quando há múltiplas regras se referindo ao mesmo elemento.

  • Box Model:

O modelo de caixa é fundamental no CSS, define como o espaço ao redor e dentro de um elemento é organizado. Cada elemento HTML é tratado como uma caixa retangular composta por margens, bordas, preenchimento e conteúdo.

  • Responsividade e Layouts:

O CSS permite criar layouts responsivos, adaptando a aparência de um site a diferentes tamanhos de tela, dispositivos e orientações.

  • Media Queries:

Permitem aplicar estilos diferentes com base em características do dispositivo, como largura da tela, densidade de pixel, orientação, etc.

  • Frameworks e Pré-processadores:

Existem frameworks CSS (como Bootstrap, Foundation) e pré-processadores (como SASS, LESS) que facilitam o desenvolvimento, oferecendo estilos predefinidos e recursos avançados.
