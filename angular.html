Introdução do Angular

Angular é uma plataforma de desenvolvimento web de código aberto criada e mantida pelo Google. Ela é utilizada para construir aplicativos web dinâmicos e interativos especialmente single-page applications (SPAs), onde a maior parte do processamento ocorre no navegador do usuário, resultando em uma experiência mais fluida e responsiva O núcleo do Angular é baseado em TypeScript, uma linguagem de programação que adiciona recursos ao JavaScript como tipos estáticos e melhorias na organização do código.
O Angular fornece um conjunto abrangente de ferramentas e recursos que facilitam o desenvolvimento, teste e manutenção de aplicativos complexos.


Versões


Atualmente, há duas versões principais do Angular:
AngularJS e Angular.

O AngularJS é a versão mais antiga, e é baseado em um modelo de desenvolvimento de MVC (Model- View-Controller).

Já o Angular é o nome da nova versão, que é uma reescrita completa do AngularJS.
A nova versão utiliza o modelo de desenvolvimento de MVC através do sistema de módulos e componentes, sendo um framework mais moderno e integrado com outras tecnologias.

Instalando o Angular

O primeiro passo para utilizar o Angular CLI é realizar a sua instalação e, para tal, é necessário ter o Node.JS instalado. Caso ainda não o tenha, acesse o site do Node.js e baixe a última versão destinada ao seu sistema operacional.
Com o Node.JS instalado, temos à disposição o comando npm, necessário para instalar o Angular CLI. Neste ponto, será preciso abrir o terminal do seu sistema operacional e digitar a instrução:
npm install -g @angular/cli
Essa instrução, normalmente, requer um bom tempo para que seja concluída, pois diversas bibliotecas do Node.JS serão obtidas através da internet.
Concluída a instalação, o Angular CLI está à nossa disposição através do comando ng, que é um programa para ser executado a partir da linha de comando e para o qual podemos passar diversos parâmetros. É dessa forma que criamos um novo projeto em Angular, criamos artefatos, realizamos o build do projeto, entre outras opções.

Verificando a instalação

Verifique a instalação global O Angular CLI pode ser instalado globalmente em seu sistema, permitindo que você crie e gerencie projetos de qualquer diretório.
Para verificar se o Angular CLI está instalado globalmente, você pode executar: ng --version Isso também mostrará a versão do Angular CLI instalada. Se você receber informações sobre a versão do Angular CLI após executar um dos comandos acima, isso significa que o Angular está instalado em seu sistema.

Criando um projeto em Angular

Fazer um projeto no Angular envolve várias etapas, desde a configuração inicial até o desenvolvimento de componentes e serviços. Use o Angular CLI: ng new NomeDoProjeto Navegar para o projeto: Acesse o diretório recém-criado: cd NomeDoProjeto Desenvolvimento Componentes: Os componentes são blocos de construção fundamentais do Angular. Você pode criar novos componentes usando o Angular CLI: ng generate component NomeDoComponente Serviços: Os serviços são usados para compartilhar lógica de negócios entre componentes. Crie um serviço usando o Angular CLI: ng generate service NomeDoServico Roteamento: Se o seu projeto envolver várias páginas ou seções, você precisará configurar o roteamento. Isso é feito no arquivo app-routing.module.ts. Templates e Estilos: Desenvolva os templates dos seus componentes usando HTML e estilize-os com CSS. O Angular permite o uso de folhas de estilo globais ou encapsulados por componente. Execução e Teste xecutar o aplicativo: Inicie o servidor de desenvolvimento usando o Angular CLI: ng serve Acesso ao aplicativo: Abra um navegador e navegue para http://localhost:4200/ para visualizar o aplicativo em execução. Implantação Compilação: Antes de implantar, você precisa compilar o aplicativo para produção: ng build --prod Implantação em Servidor: Os arquivos gerados após a compilação estão localizados na pasta dist. Você pode fazer o upload desses arquivos para um servidor web para implantar o aplicativo.
